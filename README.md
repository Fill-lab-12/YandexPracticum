Репозиторий с разбором задач из Yandex Handbook.

## Task_1.py
### Первая задача
[Открыть код](./Module%202.1/Task_1.py)

Тут всё легко: выводим заранее заданный текст с помощью встроенной функции `print()`.
`print()` может принимать строку, число или переменную.

## Task_2.py
### Вторая задача
[Открыть код](./Module%202.1/Task_2.py)

Тут тоже всё достаточно просто: сначала выводим вопрос с помощью функции `print()`.
Потом просим пользователя ввести данные: `a = input()`.
После этого выводим приветствие уже с переменной, а не с фиксированным текстом: `f"Привет, {a}"`.

Фигурные скобки `{}` внутри `f""` подставляют значение переменной в строку.
