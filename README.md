# Task_T.py
### Двадцатая задача
[Открыть код](./Module%202.1/Task_T.py)

Последняя и самая сложная задача — система уравнений! Нужно узнать, сколько килограммов каждого вида котлет в смеси.

1. Принимаем: общий вес `n`, цену за 1 кг `m`, цену за 1 кг первого вида котлет `k1`, цену за 1 кг второго вида котлет `k2`.
2. Используем систему уравнений для решения:
   - K1*x + K2*y = m*n (уравнение стоимости)
   - x + y = n (уравнение количества)
3. Решаем и получаем: `x = n * (m - k2) // (k1 - k2)` и `y = n - x`.
4. Выводим количество килограммов каждого вида.
