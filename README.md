# YandexPracticum
Репозиторий с разбором задач из Yandex Handbook.

## Task_1.py
### Первая задача
[Открыть код](./Module%202.1/Task_1.py)

Эта задача лёгкая: выводим заранее заданный текст с помощью функции `print()`.

`print()` может принимать разные типы данных:
`строку`, `число`, `переменную`.

1. В первой строке задаём текст.
2. Во второй строке выводим его в терминал.
3. В результате пользователь видим то что дали 'print()'.

## Task_2.py
### Вторая задача
[Открыть код](./Module%202.1/Task_2.py)

Эта задача впервые использует ввод данных от пользователя через `input()`.

1. Сначала выводим вопрос с помощью `print()`.
2. Затем считываем ввод пользователя: `a = input()`.
3. После этого выводим приветствие: `f"Привет, {a}"`.
4. Фигурные скобки `{}` внутри `f""` подставляют значение переменной в строку.

## Task_4.py
### Четвёртая задача
[Открыть код](./Module%202.1/Task_4.py)

Эта задача уже интереснее для новичков: нужно рассчитать сдачу по формуле `38 * 5 // 2`.

В Python есть несколько базовых операторов, которые будут часто встречаться в задачах хендбука:
`+`, `-`, `*`, `/`, `//`, `%`, `**`.

1. В первой строке узнаём, сколько денег у покупателя.
2. Во второй строке задаём формулу стоимости.
3. В третьей строке считаем сдачу.
4. В конце выводим результат через `print()`.
